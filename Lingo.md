## 初步窥探

> 1、lingo中默认是非负实数，即 x ≥ 0；
>
> 2、Lingo不区分大小写
>
> 3、所有注释应该是： ! 这里写注释 ;
>
> 4、对于非线性问题，还需要点击 Solver -> Options -> Global Solver，勾选 use Global Solver

## 常用命令

```Lingo
@bin(x) ! 表示 x 为0或者1;
@gin(x) ! 表示 x 为整数;
@free(x) ! 表示 x 为任意实数，即去除 x ≥ 0 的限制;
@bnd(1,x,u) ! 表示 x 为[1,u]之间的实数;
```

## 一个例题

```lingo
! sets是定义变量断 ;

sets:
S/1..6/: a,b,d  	;	! 定义S集合，是即一个属于S集合的变量都有6个变量，这里定义了a，即变量a有a1 a2 a3 a4 a5 a6 ; 
T/1..2/: e,x,Y	;	! 定义T集合;
U(S,T): c	;	! 定义双下标的;
endsets

! data是定义数据段 ;

data:
	a = 1.25	8.75	0.5	5.75	3	7.25	;
	b = 1.25	0.75	4.75	5	6.5	7.75	;
	d = 3		5	4	7	6	11	;
	x = 5 	2					;
	y = 1		7					;
	e = 20	20					;
enddata

!接下来写我们的模型;

!;
min = @sum(
	T(j):
		@sum(
		S(i):
			c(i,j) * @sqrt( (x(j)-a(j))^2 + (y(j)-b(i))^2 )

		)
);

! 对上述的模型进行解释 ;
!
	1、首先、求后面式子的最小值
	2、@sum()的意思是sum相当于∑求和符号，对括号里面进行求和，求和范围为T(j)
	3、@sum(范围:求和式子)	冒号前面的是范围，后面的是求和式子，然后我们例题中是有两层求和的
	4、第一层的求和，范围是 j = 1 ~ 2，所以使用 T 集合，T 集合是上述定义由 1 ~ 2 组成的一个集合名字
	5、第二层的求和，范围是 i = 1 ~ 6，所以使用 S 集合，S 集合是上述定义由 1 ~ 6 组成的一个集合名字
;


!接下来写我们的约束条件 st ;

! ∑[j=1~2](c[ij]) = d[i] ;
!	式子扩写为下面的
	c11 + c12 = d1;!
	c21 + c22 = d2;!
	c31 + c32 = d3;!
	c41 + c42 = d4;!
	c51 + c52 = d5;!
	c61 + c62 = d6;!
;

@for(
S(i):
	@sum(
	T(j):
		c(i,j)		
	) = d(i)
);

! 第二个约束条件，它是不等式约束 ;
@for(
T(j):
	@sum(
	S(i):
		c(i,j)
	) <= e(j)
);

! @for(范围:每个式子)
	1、适用于多个式子，但是写成了单个式子而已，即：xi < 32*i , i = 1,2,3...8，这个就可以用@for来写，范围是 1 ~ 8
	2、后面的每个式子应该是等式或者不等式，如上述中的 @sum(T(j):c(i,j)) = d(i) , 即对cij求和的式子需要小于d（i）
;
```
